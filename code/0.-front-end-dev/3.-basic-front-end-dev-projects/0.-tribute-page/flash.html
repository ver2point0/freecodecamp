  • All incarnations of the Flash can move, think, and react at light speeds as well as having superhuman endurance that allows them to run incredible distances.
Barry Allen

A Central City assistant police forensic investigator. Moments after an explosion at the S.T.A.R. Labs particle accelerator, Barry is struck by lightning in his laboratory and doused by chemicals affected by the accident. When he awakens from a nine-month coma, he has superhuman speed.
